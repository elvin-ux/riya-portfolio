The current hero section still does not capture the emotional quality of the approved reference.

The issue is not spacing, typography, or component structure anymore.

The issue is color atmosphere, visual weight, and emotional tone.

The client's preferred visual identity is:

* White
* Soft pastels
* Feminine
* Elegant
* Calm
* Premium
* Airy
* Editorial

The website should feel light and effortless.

It should never feel heavy, saturated, overly pink, or decorative.

The overall impression should be:

"Elegant television presenter and event host."

Not:

"Wedding invitation website."

Not:

"Beauty influencer landing page."

Not:

"Event management company."

The current implementation is still relying too heavily on a large pink background shape.

This makes the composition feel visually heavy and creates competition with the hero image.

COLOR SYSTEM

White must become the dominant visual color.

Target visual distribution:

70% White

20% Very Soft Pastel Tones

10% Accent Colors

The page should feel almost white from a distance.

Pastels should appear as atmosphere rather than as visible blocks of color.

Primary Canvas:

#FFFFFF

Secondary Surface:

#FCFAFB

Blush Tint:

#F6E8EC

Lavender Tint:

#F7F4FA

Ice Gray:

#F8FAFC

Text:

#1E1E1E

Accent Rose:

#D98C9A

The rose accent should only appear in:

* Logo
* CTA button
* Small highlights
* Active navigation state

Do not use large solid rose areas.

BACKGROUND COMPOSITION

The hero background should no longer be a single oversized pastel shape.

Instead create layered atmospheric depth.

Layer 1:

Pure white canvas.

Layer 2:

Very subtle radial glow behind the hero image.

Opacity should remain extremely low.

The visitor should feel it rather than notice it.

Layer 3:

Soft translucent pastel clouds.

Use:

* blush
* lavender
* bone white

These should overlap naturally.

No obvious geometric shapes.

No circles.

No blobs.

No hard edges.

Layer 4:

Subtle paper-like texture.

Extremely soft.

The goal is to make the page feel tactile and premium.

Layer 5:

Very fine botanical line illustrations.

Placed only near the lower-right area.

Keep opacity below 5%.

The user should discover them rather than immediately notice them.

HERO IMAGE COMPOSITION

The transparent cutout image is now the primary visual asset.

Treat Riya as the hero composition itself.

Do not make the background the hero.

Do not make decorative shapes the hero.

The hero is Riya.

Scale the cutout up significantly.

The image should command attention immediately.

Current implementation still feels too small.

Increase visual dominance by approximately 25%.

Allow portions of the image to overlap atmospheric background layers.

The subject should feel naturally integrated into the environment.

The cutout should appear elegant and effortless.

Not pasted.

Not floating.

Not isolated.

The composition should feel art directed.
